Горячие клавищи:

	CTRL+C
	
		Для остановки процесса через консоль.

	CTRL+~
	
		Открыть/закрыть консоль.

	CTRL+L
	
		Очистить консоль.

	CTRL+SHIFT+~
	
		Открыть новую консоль.

Дополнения:

	Material Icon Theme

		Установка иконок в зависимости от имен проектов.

	Path Intellisense


		Автодополнение путей.

	Prettier - Code formatter

		Стилизация кода.

	ESLint

		Линтер для "Javascript".

Работа в "оффлайн" режиме. При работе без доступа к интернету, при запуске проекта может отобразиться ошибка в файле "package.json" (при его наличии). Данная ощибка не критична, и указывает на то, что редактор не может загрузить стили
для файла "package.json". Можно игнорировать данную ошибку и продолать работу. Если нужно устранение данной ошибки, то можно найти решение данной проблемы по ссылке "stackoverflow.com/questions/47691993".
